POE2 item value triage

A simple POE2 item value triage flow for deciding what to inspect, keep, price-check, or ignore.

View audit credits Back to POE2 Lab

Short answer

Do not try to price every drop. Triage items by build relevance, base type, useful affix combinations, required level, and whether there is actual trade demand. A fast decision flow is more useful than perfect price knowledge.

DecisionKeep looking ifMove on if
Build relevanceIt matches a popular skill, weapon, or defense need.It has random stats with no clear user.
Base qualityThe base type is desirable for current builds.The base is weak for its level or slot.
Affix groupSeveral useful stats work together.Good stats fight each other or support no plan.
LiquiditySimilar items appear to sell.Only fantasy listings exist.
Personal useIt improves your current build now.It would sit in stash indefinitely.

Checklist

  • Sort drops into use, price-check, craft-test, and ignore buckets.
  • Do not stash every maybe item.
  • Check whether the item solves a real build problem.
  • Watch for affixes that work together, not isolated high numbers.
  • Review stale stash tabs weekly.

Example

An item with one impressive stat may still be worthless if the rest of the item does not support a real build or trade buyer.

Common mistakes

  • Keeping everything because it might be useful later.
  • Overpricing items that never sell.
  • Ignoring build context when judging affixes.
  • Confusing listed prices with completed demand.
