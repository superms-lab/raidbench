# Duo Raid Roles

Assign roles before starting.

| Role | Player | Job |
| --- | --- | --- |
| Boom lead | | Places boom and calls next target |
| Security | | Watches counters and flank angles |
| Seal/loot | | Carries seal materials and moves loot |

Shared rules:

- One player always knows the exit.
- One player watches while the other loots.
- Loot moves in waves, not only at the end.
