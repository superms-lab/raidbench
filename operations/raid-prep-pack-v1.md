## 1. Raid Target Score

Score a target before farming sulfur.

| Question | Score |
| --- | ---: |
| Base looks active but not heavily defended | +2 |
| Door path is visible or likely short | +2 |
| Target has recent signs of loot movement | +2 |
| Base has compound/external TCs/turrets | -3 |
| Raid requires wall-side entry without scouting | -2 |
| You cannot bring one extra layer of boom | -3 |

Rule:

```text
Score 4+ = consider raiding
Score 1-3 = scout more
Score 0 or lower = skip
```

## 3. Solo Raid Checklist

Before leaving base:

- Confirm target path and escape route.
- Bring enough boom for one extra layer.
- Bring building plan and basic seal materials.
- Bring meds, ammo, and one fallback weapon.
- Bank sulfur in smaller trips before crafting everything.
- Decide stop condition before starting.

Stop if:

- You hit an unexpected armored layer.
- Counters arrive before first breach.
- Loot value no longer justifies extra boom.

## 4. Duo Raid Roles

Player A:

- Places boom.
- Tracks raid path.
- Calls next target.

Player B:

- Watches counters.
- Carries seal materials.
- Controls loot transfer.

Shared rule:

```text
One player should always know the exit path.
```
